このリポジトリは 2026/6/6, 7, 13, 14 の4日間に開催される jsys 26 ネットワーク部門研修の資料です。

## 内容

- Web フロントエンド
- Web バックエンド

の両方に対応しており、
Web開発やプログラミングやコンピュータの経験、それぞれの興味に応じて、
どちらかのコースを選択して進めてもらうことを想定しています。

各コースの各日程の内容は、より小さいフェーズに分割されています。
特に経験がない人は上から順に読み進めることを想定していますが、
最初の方の内容は経験者にとってはすでに知っていて退屈かもしれません。
自分が分かるところまで適宜飛ばして進めてもらって構いません。
とにかく、おもしろそうなところを手を動かしながら読むことを強く推奨します。

- frontend
  本資料は 3 日間かけて Web プログラミングの基礎の学習から、モダンなフロントエンド開発の体験までを行うことを想定しています。各日程の内容はフェーズ 1、フェーズ 2…… と分けられ、それらは各日での学習内容によって分けられています。資料の内容はおおよそ次の通りです。
  - day1
    - phase1
      - フェーズ 1 では HTML に入門し、HTML を用いた簡単な Web ページの作成を体験します。
    - phase2
      - フェーズ 2 では CSS を学び、素の HTML ドキュメントを装飾する方法について学習します。
    - phase3
      - フェーズ 3 では JavaScript を勉強し、HTML に組み込んで使用する方法について学習します。また、学んだことを応用して DOM の操作や fetch API を活用した実践的な JavaScript プログラミングを行います。
  - day2
    - phase0
      - フェーズ 0 では、基本的なシェル操作を学習します。また、本講習で使用するアプリケーションのインストールや環境構築を行います。
    - phase1
      - フェーズ 1 では、TypeScript を用いて静的型付け言語を習得し、堅牢なプログラムを書くことを学びます。
  - day3
    - phase1
      - フェーズ 1 では React の基本を学んだのち、React を用いて実用的なアプリケーションを作成します。

- backend
  バックエンドコースでは、
  Web開発の経験がない人でも取り組めるように、「バックエンドって何をしているの?」から始めて、匿名掲示板のようなWebアプリを作ることを目指します。
  ただ、なんらかのプログラミング言語は触ったことがあるくらいの経験値を期待しているのでちょっぴり難しいかもしれません。
  フロントエンドコースでTypeScriptまでやったのでこっちを見てみるというのもありでしょう。
  - 1章: 知っておきたい! 知識編
  - 2章: 楽しい、楽しい 実践編
  - 3章: 高度な話題! 発展編
