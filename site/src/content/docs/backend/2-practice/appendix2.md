「2章: 楽しい、楽しい 実践編」では、
「匿名掲示板」を題材にしたテンプレートリポジトリで実際に作業をします。

バックエンドの動作を検証するのに、`curl`というコマンドが最も原始的でシンプルということで本文で紹介しています。

しかし、`curl`は原始的すぎて辛い部分もあり、
HTTPを確認するだけならGUIでぐいぐいやる方法とかあるんでは?と思われるかもしれません。

実は、あります。

- ブラウザの開発者ツール (ネットワークタブ)
- [HTTPie](https://httpie.io/)
- [Postman](https://www.postman.com/)

また、今回は「匿名掲示板」の検証用フロントエンドを
テンプレートリポジトリの`frontend/`以下に作成しました。

以下のように、リポジトリrootから`frontend/`以下に移動し、
`npm i`で必要なパッケージをインストールした後、
`npm run dev`でフロントエンドの開発サーバを起動できます。

[http://localhost:5173](http://localhost:5173) から検証用フロントエンドを使うことが出来ます。

```sh
$ cd frontend/
$ npm i
$ npm run dev
```
