## このサイトについて

このサイトは **2025/6/21, 22 に開催される IPC / jsys 合同 Web 研修 2025** の研修資料を、Astro Starlight で再構成した非公式ドキュメントサイトです。Web 開発をしたことがない 1 年生を主な対象としており、3 日間で **HTML / CSS / JavaScript** の入門から、**React + TypeScript** によるモダンな Web 開発までを体験することを目標とします。

## 進めかた

各日程の内容は **フェーズ 1 / フェーズ 2 …** に分割されており、上から順に読み進めることを想定しています。手を動かしながら読むことを強く推奨します。左サイドバーから直接フェーズ単位の章へ飛ぶこともできます。

PDF で配布されている資料（Day 1 Phase 2 / Day 2 Phase 0・1）は、ページ内に PDF ビューアを埋め込んでありますが、ブラウザによっては別タブで開いた方が読みやすいことがあります。
