## このサイトについて

このサイトは 2026/6/6, 7, 13, 14 の4日間に開催される jsys 26 ネットワーク部門研修の資料です。

- Web フロントエンド
- Web バックエンド
の両方に対応しており、
Web開発やプログラミングやコンピュータの経験、それぞれの興味に応じて、
どちらかのコースを選択して進めてもらうことを想定しています。

各コースの各日程の内容は、より小さいフェーズに分割されています。
特に経験がない人は上から順に読み進めることを想定していますが、
最初の方の内容は経験者にとってはすでに知っていて退屈かもしれません。
自分が分かるところまで適宜飛ばして進めてもらって構いません。
とにかく、おもしろそうなところを手を動かしながら読むことを強く推奨します。
左サイドバーから直接フェーズ単位の章へ飛ぶこともできます。

PDF で配布されている資料（フロントエンド Day 1 Phase 2 / Day 2 Phase 0・1）は、ページ内に PDF ビューアを埋め込んでありますが、
ブラウザによっては別タブで開いた方が読みやすいことがあります。

### バックエンド

Web開発と言ったときに、ブラウザに表示されるインタラクティブできれいなUIを作ることを想像する人が多いかも知れません。

しかし、SNSやオンラインショッピングサイト、メッセージングツールのWeb版...というようなWebサービスは、
ブラウザで動くフロントエンドだけで実現されているのではありません。

ユーザからは見えないどこかにあるサーバで動いている、バックエンドとよばれるソフトウェアの部品が、
ユーザや様々なモノのデータを管理し、ユーザの必要に応じて提供することで、Webサービスは動いているのです。

一見地味に思えますが、縁の下の力持ち的な重要な役割を占めているのです。

そんな「Webバックエンド」の開発に、興味は湧きませんか?

バックエンドコースでは、
Web開発の経験がない人でも取り組めるように、「バックエンドって何をしているの?」から始めて、匿名掲示板のようなWebアプリを作ることを目指します。
ただ、なんらかのプログラミング言語は触ったことがあるくらいの経験値を期待しているのでちょっぴり難しいかもしれません。
フロントエンドコースでTypeScriptまでやったのでこっちを見てみるというのもありでしょう。
