## このページの構成

| 章 | 内容 |
| --- | --- |
| [1章: Git/GitHubとは](/git/1what-is-git/) | Gitが何のためのツールか、なぜ使うのかを理解し、インストールと初期設定を済ませます。 |

このページでは、プロジェクトの開発でほぼ必ず使うことになるバージョン管理ツール「Git」の基本を学びます。
「ファイルのバージョンをどう管理するか」という身近な課題から出発し、複数人でのチーム開発で使われる標準的なワークフローを一通り体験できることを目標にしています。

1〜4章の練習には、[twin-te/git-practice-2026](https://github.com/twin-te/git-practice-2026)を使います。ここで基本操作を身につけたら、仕上げの5章では**本物のTwin:te公式サイトのリポジトリ**にPull Requestを出し、先輩のレビューを受けてマージされ、[www.twinte.net](https://www.twinte.net) にあなたが載ります。

いずれもTwin:teのGitHub organization(twin-te)への招待が必要なので、まだの人は運営に声をかけてください。
