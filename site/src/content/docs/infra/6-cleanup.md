title: "6. 後片付け"
---

**この演習で一番大事なステップです。**
[はじめに](/infra/)に書いた通り VM は時間課金なので、消し忘れがこの演習における最大の事故です。必ず以下を実行してください。

1. Akamai Cloud でインスタンス(`training-<あなたの名前>`)の右のメニューから **Delete** を選び、確認ダイアログに従って削除する
2. **Firewalls** から `training-<あなたの名前>-fw` も削除する
3. **DNS レコードと OAuth コールバック URL の削除**をTAに依頼する
4. Linodes の一覧から自分のインスタンスが**消えている**ことを確認する

DNS レコードを消す理由は、お金ではなく**セキュリティ**です。VM を削除すると、あなたが使っていた IP アドレスはクラウド事業者に返却されて、いずれ**他の誰か**に割り当てられます。DNS レコードを残したままだと、設定したドメインがその他人のサーバを指すことになります(ぶら下がったままの DNS、**dangling DNS** と呼びます)。twinte.net の名前で他人がコンテンツが配信できてしまい、乗っ取りやフィッシングなどに悪用される危険があります。

TAにチェックしてもらったらインフラ編の演習は完了です。おつかれさまでした!
この演習では、自分の PC で動いていたアプリをサーバに引っ越し、ファイアウォールで守り、本番と同じ方式で動かし、本物の URL で公開しました。
